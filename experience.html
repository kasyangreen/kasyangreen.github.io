Time Series Analysis of Mangrove in Myanmar


This project used Google Earth Engine to map Mangrove deforestation in Myanmar since 1998, just before the Ramsar Convention on Wetlands, and the current day using a Mangrove index.


Mangrove forests are unique because they can be differentiated from other terrestrial forests using remote sensing. This is done using a wetness index (MNDWI), to detect moisture, and a vegetation index (NDVI), to detect canopy. When you combine these, it’s possible to create a “Mangrove Index” (MMRI). These three indexes can be combined to create a color composite showing mangrove forest in bright green (Composite RGB = Wetness Index, Vegitation Index, Mangrove Index).
